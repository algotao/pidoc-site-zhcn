# SSD Kit (SSD套件/新品)

## 关于

<img src="/img/accessories/ssd-kit/ssd-kit.png" alt="SSD套件" width="80%" />
*512GB Raspberry Pi 固态硬盘套件*

Raspberry Pi 固态硬盘套件捆绑了一个 [Raspberry Pi M.2 HAT+](./m2-hat-plus.mdx) 和一个 [Raspberry Pi SSD](./ssds.mdx)。

Raspberry Pi SSD 套件包括一个 16 毫米堆叠接头、垫片和螺丝，以便与 Raspberry Pi Active Cooler 一起安装在 Raspberry Pi 5 上。

## 安装

要安装树莓派固态硬盘套件，请按照 [树莓派M.2 HAT+安装说明](./m2-hat-plus.mdx#安装) 操作。
