# 键盘和鼠标

## 键盘使用入门

我们的官方键盘包含三个主机 USB 端口，用于连接外部设备，如 USB 鼠标、U 盘和其他 USB 控制设备。

产品的微型 USB 端口用于连接树莓派。通过键盘内置的 USB 集线器，树莓派可控制三个 USB A 型端口并为其供电。

<img src="/img/accessories/keyboard-and-mouse/back-of-keyboard.png" alt="键盘背面" width="80%" />

### 键盘功能
树莓派键盘有三个锁定键： `Num Lock` (数字锁定)、`Caps Lock` (大写锁定) 和 `Scroll Lock` (滚动锁定)。右上角有三个 LED 指示灯，可显示哪些锁已启用。

<img src="/img/accessories/keyboard-and-mouse/num-cap-scroll.png" alt="键盘锁定" width="70%" />

#### 数字锁定
允许在字母键上使用红色数字键，从而有效创建数字小键盘。按`Num Lock`键可启用或禁用该模式。

#### 大写锁定
允许输入大写字母；在此模式下按 Shift 键可输入小写字母。按下`Caps Lock`键可启用或禁用该模式。

#### 滚动锁定（ScrLk）
允许在不使用鼠标的情况下使用光标键浏览网页和电子表格。按住 Fn 键的同时按下 `Scroll Lock` 键可启用或禁用该模式。

## 鼠标使用入门
我们的官方鼠标有三个按钮，可激活高质量的微动开关。滚轮用于浏览文档和网页时快速滚动。

<img src="/img/accessories/keyboard-and-mouse/the-mouse.png" alt="鼠标" width="70%" />
使用鼠标时，请始终将其放在平坦、稳定的表面上。鼠标会光学检测放置表面的移动。在无特征的表面上，例如 PVC 或亚克力桌面，鼠标无法检测到移动。在这种表面上工作时，请将鼠标放在鼠标垫上。

## 连接到一起
这是我们推荐的将树莓派、官方键盘和集线器以及官方鼠标连接在一起的配置。键盘上的集线器可确保轻松访问 USB 驱动器，鼠标线缆整齐，同时足够长，让你可以左右手使用鼠标。

<img src="/img/accessories/keyboard-and-mouse/everything.png" alt="连接到一起" width="70%" />

:::note[注意]
电源必须连接到树莓派，键盘必须连接到树莓派。如果电源连接到键盘，而树莓派通过键盘供电，那么键盘将无法正常工作。
:::
