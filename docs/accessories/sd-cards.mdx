# SD 卡

## 关于

<img src="/img/accessories/sd-cards/sd-hero.jpg" alt="SD卡" width="80%" />
*一张插入 Raspberry Pi 5 的 Raspberry Pi SD 卡*

SD 卡的质量是决定 Raspberry Pi 整体用户体验的关键因素。总线速度慢和缺乏命令队列会降低性能，即使是最强大的 Raspberry Pi 型号也不例外。

Raspberry Pi 的官方 microSD 卡支持 DDR50 和 SDR104 总线速度。此外，Raspberry Pi SD 卡支持命令队列（CQ）扩展，允许随机读取操作流水线化，确保最佳性能。

您甚至可以购买预编程了最新版 Raspberry Pi 操作系统的 Raspberry Pi SD 卡。

Raspberry Pi SD 卡有以下大小：

* 32GB
* 64GB
* 128GB

## 规格

128GB Raspberry Pi SD 卡
<img src="/img/accessories/sd-cards/sd-cards.png" alt="SD卡" width="80%" />

Raspberry Pi SD 卡使用 SD6.1 SD 规范。

Raspberry Pi SD 卡使用 microSDHC/microSDXC 外形。

Raspberry Pi SD 卡的速度等级如下： C10、U3、V30、A2。

下表描述了 Raspberry Pi SD 卡使用 4KB 随机数据的读写速度：

| Raspberry Pi 型号 | 接口 | 读取速度 | 写入速度 |
| -- | -- | -- | -- |
| 4 | DDR50 | 3,200 iops | 1,200 iops |
| 5 | sdr104 | 5,000 iops | 2,000 iops |
