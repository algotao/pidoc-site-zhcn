# 保护底座

## 关于

<img src="/img/accessories/bumper/bumper.jpg" width="80%" alt="用于 Raspberry Pi 5 的 Raspberry Pi 保护底座" />
*用于 Raspberry Pi 5 的 Raspberry Pi 保护底座*

用于 Raspberry Pi 5 的 Raspberry Pi 保护底座是一个卡入式硅胶套，可保护电路板的底部和边缘。安装后，Raspberry Pi 的安装孔仍可通过保护底座进入。

保护底座仅与 Raspberry Pi 5 兼容。

## 装配说明

<img src="/img/accessories/bumper/assembly.png" width="80%" alt="组装保护底座" />
*组装保护底座*

将 Raspberry Pi 保护底座安装到 Raspberry Pi 上：

1. 关闭 Raspberry Pi 并断开电源线。
2. 从 Raspberry Pi 的 SD 卡插槽中取出 SD 卡。
3. 将保护底座与电路板对齐。
4. 将电路板轻轻但牢固地压入缓冲器，注意避免缓冲器与电路板的任何组件接触。
5. 将 SD 卡插回 Raspberry Pi 的 SD 卡插槽。
6. 重新连接 Raspberry Pi 的电源。

从 Raspberry Pi 上取下 Raspberry Pi保护底座：

1. 关闭 Raspberry Pi 并断开电源线。
2. 从 Raspberry Pi 的 SD 卡插槽中取出 SD 卡。
3. 轻轻但用力地将保护底座从电路板上剥离，注意避免保护底座与电路板的任何组件接触。
4. 将 SD 卡插回 Raspberry Pi 的 SD 卡插槽。
5. 重新连接 Raspberry Pi 的电源。
