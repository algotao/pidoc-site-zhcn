# SSD

## 关于

<img src="/img/accessories/ssds/ssd.png" alt="SSD" width="80%" />
*512GB Raspberry Pi 固态硬盘*

固态硬盘的质量是决定 Raspberry Pi 整体用户体验的关键因素。
Raspberry Pi 提供经过测试的官方固态硬盘，以确保与 Raspberry Pi 型号和外设的兼容性。

Raspberry Pi 固态硬盘有以下大小：

* 256GB
* 512GB

要在 Raspberry Pi 上使用固态硬盘，你需要一个兼容 Raspberry Pi 5 的 M.2 适配器，例如 [Raspberry Pi M.2 HAT+ 或 M.2 HAT+ Compact](./m2-hat-plus.mdx)。

## 规格

Raspberry Pi 固态硬盘符合 PCIe Gen 3 标准。

Raspberry Pi 固态硬盘使用 NVMe 1.4 寄存器接口和命令集。

Raspberry Pi 固态硬盘使用 M.2 2230 外观尺寸。

下表描述了 Raspberry Pi SSD 使用 4KB 随机数据的读写速度：

| 大小 | 读取速度 | 写入速度 |
| -- | -- | -- |
| 256GB | 40,000 iops | 70,000 iops |
| 512GB | 50,000 iops | 90,000 iops |
