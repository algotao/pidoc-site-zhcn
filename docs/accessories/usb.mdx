# USB配件

树莓派提供以下高品质、可靠的USB配件，适用于树莓派及其他计算机：

* 树莓派 [集线器]，用于扩展USB连接能力。
* 树莓派 [闪存盘]，用于便携式存储。

## USB3集线器

树莓派USB3集线器通过将单个USB-A接口扩展为四个USB-A接口，为您的设备提供额外连接能力。通过USB3集线器连接的设备，无论连接至树莓派设备还是其他计算机，均会被识别为直接连接至主机系统。

采用总线供电时，USB3集线器可从上游USB端口获取最高900毫安（4.5瓦）的电力。这4.5瓦功率由四个下游USB 3.0端口共享，足以满足低功耗外设（如大多数鼠标和键盘）的供电需求。

为支持高功耗外设及设备，USB 3集线器另配备可选USB-C电源输入接口，可在需要时为下游端口提供外部电源。

### USB3集线器特性
树莓派USB3集线器提供USB扩展功能，具备高速数据传输、灵活供电选项，兼容USB 3.0与USB 2.0接口。相关核心规格详见下表。

*树莓派USB3集线器功能及对应规格*

| 功能 | 规格 |
| -- | -- |
| *USB扩展* | 提供四个下游USB 3.0 Type-A端口；配备8厘米固定线缆的上游USB 3.0 Type-A接口。 |
| *传输速率* | 支持最高5Gb/s数据传输速率。 |
| *供电能力* | 从连接设备获取电源时，可提供最高5V/900mA（4.5W）供电；通过可选USB-C电源输入自供电时，可提供最高5V/3A（15W）供电。 |
| *兼容性* | 兼容USB 3.0 Type-A接口，向下兼容USB 2.0标准。 |

## 闪存盘

树莓派闪存盘是一款USB 3.0存储设备，可提供便携式存储功能，用于文件传输、创建备份或运行操作系统。该闪存盘提供两种存储容量（128 GB或256 GB），采用非易失性闪存技术，断开连接后仍能保留数据。

<img src="/img/accessories/usb/flash-drive-hero.png" width="80%" alt="树莓派闪存盘" />
*树莓派闪存盘*

### 闪存盘特性

树莓派闪存盘采用轻量化便携设计，金属外壳可挂载钥匙圈或挂绳。两种容量均提供可靠的移动存储性能，兼容USB 3.0与USB 2.0接口。相关核心规格详见下表。

*树莓派闪存盘特性与对应规格*

| 特性 | 规格 |
| -- | -- |
| *紧凑机身* | 重量低于6克；具体尺寸参见 [闪存盘尺寸](#闪存盘尺寸)。 |
| *存储容量* | 提供128 GB和256 GB两种容量，采用非易失性闪存。 |
| *性能表现* | 随存储容量变化。详情参见 [性能](#闪存盘性能)。 |
| *兼容性* | 支持USB 3.0 Type-A接口，向下兼容USB 2.0。 |

该闪存盘具备高每秒输入/输出操作次数（IOPS），可快速访问小文件。同时为操作系统启动、应用程序运行等计算任务提供响应式性能。下表根据128 GB与256 GB两种存储容量选项，总结了树莓派闪存盘的关键性能特征。

### 闪存盘性能

*树莓派闪存盘性能特征（按存储容量划分）*

| 性能特征 | 128 GB | 256 GB |
| -- | -- | -- |
| **4 kB随机读取** | 16,000 IOPS | 18,000 IOPS |
| **4KB随机写入* | 21,000 IOPS | 22,000 IOPS |
| *顺序写入速度（持续）* | 75 MB/s | 150 MB/s |
