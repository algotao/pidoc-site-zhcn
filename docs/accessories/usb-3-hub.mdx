# USB HUB

## 关于

[Raspberry Pi USB 3 集线器](https://www.raspberrypi.com/products/usb-3-hub/) 为您的设备提供额外的连接，将一个 USB-A 端口扩展为四个。可选的外部 USB-C 电源输入支持高功率外设。您可以使用 USB 3 集线器为低功耗外设供电，如大多数鼠标和键盘，无需外接电源。

<img src="/img/accessories/usb-3-hub/usb-3-hub-hero.png" alt="树莓派 USB 3.0 集线器" width="80%" />
*树莓派 USB 3.0 集线器*

## 规格

* 1 个上游 USB 3.0 Type-A 公连接器，带 8 厘米缆绳
* 4 个下游 USB 3.0 Type-A 端口
* 数据传输速度高达 5Gbps
* 功率传输最高可达 900 mA（4.5 W）；可选的外部 USB-C 电源输入可为高功率下游外设提供最高 5V @ 3A 的电压
* 兼容 USB 3.0 和 USB 2.0 Type-A 主机端口
