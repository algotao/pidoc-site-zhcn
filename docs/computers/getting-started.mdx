# 入门

<iframe width="80%" height="400" src="https://www.youtube.com/embed/CQtliTJ41ZE" frameBorder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowFullScreen></iframe>

树莓派（Raspberry Pi）是一种小型计算机，通常有两种主要用途：作为**台式计算机**或作为**无显示器（远程）计算机**。有关这些选项的更多信息，请参阅[设置您的树莓派](#设置您的树莓派)。

树莓派设备分为以下几个系列：

* **树莓派单板计算机 (SBC)。** 经典或**旗舰**款树莓派设备，可像台式电脑一样连接显示器、键盘和鼠标。
* **树莓派 Zero。** 旗舰款树莓派设备的精简版，USB 接口较少且功耗更低。
* **树莓派键盘电脑。** 计算机功能集成于键盘外壳中的设备，设计用于连接显示器和鼠标。
* **树莓派计算模块。** 专为工业和嵌入式应用设计的树莓派计算机小型系统级模块版本。
* **树莓派 Pico。** 一款采用树莓派微控制器（RP2040 或 RP2350）的微控制器开发板。

本页面重点介绍 **旗舰单板计算机**，但其中大部分信息也适用于树莓派 **Zero** 和 **键盘一体机**。有关树莓派全系列计算机设备的详细信息，请参阅 [树莓派计算机硬件](./raspberry-pi.mdx)。

## 硬件要求

要设置树莓派设备，您需要以下硬件。这些组件应按下列顺序准备并组装：

* 已安装操作系统的 [启动介质](#启动介质)。
* 对于桌面设置，需准备 [设备](#设备) 中列出的输入和输出外设。
* 对于无显示器设置，需另一台计算机和网络连接。
* 电源。有关指导，请参阅 [电源](#电源)。

此外，您还可以使用：

* 树莓派机箱。这有助于保护您的树莓派设备免受损坏，或防止其放置在金属表面时发生短路。
* 一根以太网（RJ45）线缆，用于将树莓派连接到有线网络。若连接无线网络则无需此线缆，您可通过输入 SSID 和密码连接 Wi-Fi。

### 启动介质 {#启动介质}

要作为计算机运行，树莓派需要一个包含 **操作系统 (OS) 镜像** 的非易失性存储设备。这种存储设备称为 **启动介质**，最常见的类型是 **microSD 卡**。某些较新的树莓派型号可以从其他介质启动，例如 **USB 大容量存储**、**网络启动 (PXE)** 以及 **通过 PCIe 接口的 NVMe 存储**。

在组装树莓派之前，您必须配置好启动介质，除非您在树莓派 4B、5、500 或 500+ 上使用网络安装，这种情况下，您将直接使用树莓派本身来配置启动介质。

无论您选择哪种存储设备作为启动介质，它都必须具备足够的存储空间和速度，以充当存储操作系统镜像的硬盘。操作系统镜像是一种预打包的软件，将其安装到启动介质上后，即可将您的树莓派从一块空白电路板转变为一台可正常运行的计算机。

有关配置树莓派启动介质的分步指南，请参阅 [将操作系统安装到启动介质](#将操作系统安装到启动介质上)。

读写速度更快的存储设备能提升树莓派的整体性能，使其运行更流畅。如果您不确定该购买哪种 SD 卡，建议考虑 [官方树莓派 microSD 卡](https://www.raspberrypi.com/products/sd-cards/)。您可能还需要一个 SD 卡读卡器，以便将 microSD 卡连接到计算机上安装操作系统。

<img src="/img/computers/getting-started/peripherals/microSD.png" alt="一张 Raspberry Pi microSD 卡。" width="50%" />
*官方 128 GB Raspberry Pi microSD 卡*

自 Raspberry Pi Model B+ 以来，所有型号均配备 microSD 卡插槽；这些 Raspberry Pi 设备会自动从插入的、带有有效操作系统镜像的 microSD 卡启动。

如果您拥有树莓派键盘电脑（树莓派 400、500 或 500+），或者您的树莓派电脑是套件的一部分，则操作系统已预先配置。您可以通过 *控制中心* 或在终端中使用 `raspi-config` 来更改配置。有关说明，请参阅 [配置](./configuration.mdx)。

#### 推荐的最低存储空间要求

所需的启动介质存储空间取决于您安装的操作系统版本中包含的软件数量。

以下建议基于您选择的操作系统足以满足入门需求，但我们建议选择更大的存储空间以留出扩展余地：

* **Raspberry Pi OS**（带基础软件的桌面版）：至少 32 GB 存储空间的 SD 卡。
* **Raspberry Pi OS Full**（带基础软件和推荐应用程序的桌面版）：至少 32 GB 存储空间的 SD 卡。
* **Raspberry Pi OS Lite**（仅限命令行、适用于无显示器设置的版本）：至少 8 GB 存储空间的 SD 卡。

其他操作系统可能有不同的存储要求，因此请务必查阅您计划安装的操作系统文档。

#### 最大支持容量

您可以使用任何容量小于 2 TB 的 SD 卡。容量超过 2 TB 的卡不被支持，因为它们超出了主引导记录（MBR）的限制。MBR 是 SD 卡上使用的分区系统，它会告诉树莓派在哪里找到操作系统，以便其启动。

部分较旧的树莓派设备仅支持从 256 GB 或更小的启动分区启动。这些设备包括：

* 树莓派 Zero。
* 第一代旗舰树莓派设备。
* 搭载 BCM2836 SoC 的早期树莓派 2 型号。

#### 插入 microSD 卡
在将 microSD 卡插入树莓派之前，必须先对其进行配置。有关配置树莓派启动介质的分步指南，请参阅 [将操作系统安装到启动介质上](#将操作系统安装到启动介质上)。

在为 microSD 卡配置好操作系统后，将其插入 microSD 卡槽，此时标签面应背向树莓派。
<img src="/img/computers/getting-started/peripherals/sd-card.png" alt="将 microSD 卡插入树莓派。" width="80%" />
*将 microSD 卡插入树莓派*

对于树莓派 3 及后续型号，以及所有树莓派 Zero 型号，microSD 卡插入后会自动卡住（无咔嗒声），取出时只需轻轻拉出。较旧的型号则需要先按压 microSD 卡以解锁。

### 设备

**无显示器设置**仅需另一台具备网络连接的计算机。更多信息请参阅 [无显示器设置](#无显示器设置)。

根据您的设备和具体使用场景，大多数 *桌面设置* 需要输入和输出外设（如键盘、鼠标和显示器）。您还必须准备相应的线缆，用于将这些外设连接至树莓派。

若要连接显示器，树莓派 4B、400、5、500 和 500+ 需要 micro-HDMI 转 HDMI 线缆；树莓派 Zero 和 Zero 2 则需要 mini-HDMI 转 HDMI 线缆。您也可以使用转接头。

#### 输入外设（键盘和鼠标） {#输入外设}

您可以使用树莓派上的任意USB端口连接[有线键盘](https://www.raspberrypi.com/products/raspberry-pi-keyboard-and-hub/)和[有线鼠标](https://www.raspberrypi.com/products/raspberry-pi-mouse/)。如果您希望将高速 USB 3.0（蓝色）端口留给其他高速外设使用，请使用树莓派上的 USB 2.0（黑色）端口连接外设。

<img src="/img/computers/getting-started/peripherals/cable-key.png" alt="将键盘连接到树莓派。" width="40%" />
*将键盘连接到树莓派*

<img src="/img/computers/getting-started/peripherals/cable-mouse.png" alt="将鼠标连接到树莓派。" width="40%" />
*将鼠标连接到树莓派*

如果您使用的是官方树莓派键盘或树莓派键盘电脑，可以将有线鼠标连接到背面的 USB 端口之一。这样可以保留树莓派设备上更多的 USB 端口。

##### 树莓派 Zero 的 USB 设备

树莓派 Zero 机型使用微型 USB 接口而非标准尺寸 USB 接口，因此需要一个微型 USB OTG 转接器。其中一个微型 USB 接口标有 PWR IN 字样，仅用于电源输入，但您可以使用任何标有 USB 的接口。

树莓派 Zero 设备无法同时为键盘和鼠标提供足够的电源（除非键盘带有内置触控板）。因此，您需要将一个带电源的 USB 集线器插入连接至树莓派 Zero 的微型 USB OTG 转接器中，随后再将鼠标连接至该 USB 集线器。

:::note
在树莓派 Zero 启动后插入 USB 设备可能会导致电压骤降，从而引发 CPU 重启。因此，我们建议您使用带电源的 USB 集线器，或者在启动前连接 USB 设备。
:::

##### 无线输入设备

如果您想使用无线键盘或鼠标，建议先使用有线键盘和鼠标完成树莓派的首次配置和组装后再进行设置。之后您可以选择以下任一方式：

* 配备 USB 蓝牙接收器的键盘或鼠标。将键盘或鼠标的 USB 适配器插入树莓派的 USB 端口即可连接。
* 带有内置蓝牙功能的键盘或鼠标，用于连接树莓派。如果您的树莓派型号支持蓝牙，请将键盘与树莓派配对。


具体操作指南，请参阅 **配置** 页面中的 [连接蓝牙外设](./configuration.mdx#连接蓝牙外设)。有关支持蓝牙的树莓派型号列表，请参阅 [网络连接](#网络连接)。

#### 输出外设（显示与音频） {#输出外设}
若将树莓派设置为台式电脑，则需要连接显示器以便查看操作内容。此外，您还可以连接扬声器或耳机以实现音频输出。如果您使用的是官方的 [树莓派显示器](https://www.raspberrypi.com/products/raspberry-pi-monitor/)，可以通过显示器背面的 3.5 毫米音频接口连接扬声器或耳机。

##### 显示器
您可以利用树莓派上的显示输出接口连接显示器。如果您的树莓派拥有多个 HDMI 接口，请将主显示器插入标有 *HDMI0* 的接口。

:::note
显示器通常需要独立的电源。如果您使用的是官方树莓派显示器，可以通过连接到树莓派的 USB 电源线为显示器供电。当您打开树莓派电源时，它会自动为树莓派显示器供电。
:::

在树莓派 4 及后续机型上，两个 HDMI 输出端口均支持最高 4K 分辨率，但具体刷新率因型号而异。

<img src="/img/computers/getting-started/peripherals/cable-hdmi.png" alt="将微型 HDMI 线缆插入树莓派。" width="80%" />
*将微型 HDMI 线缆插入树莓派*

##### 音频

根据您的树莓派型号，您可以通过 HDMI、USB、蓝牙或 3.5 毫米音频接口输出音频。您还可以通过官方树莓派显示器背面的 3.5 毫米音频接口连接音频。

* 所有配备 HDMI、微型 HDMI 或迷你 HDMI 接口的树莓派型号均支持通过 HDMI 输出音频。
* 所有树莓派型号均支持通过 USB 输出音频。
* 所有配备蓝牙功能的树莓派型号均支持蓝牙音频。
* 树莓派 1、2、3 和 4 的所有版本均配备 3.5 毫米辅助 [TRRS](https://en.wikipedia.org/wiki/Phone_connector_(audio)#TRRS_standards) 接口，可能需要放大才能获得足够的输出音量。TRRS 接口提供线路电平输出，而非经过放大的扬声器电平输出。


### 电源供应 {#电源}
将树莓派连接至电源是硬件设置的最后一步。我们建议使用适合您所用树莓派型号的官方树莓派电源适配器。

Raspberry Pi 在连接电源后会立即启动。除非您已 [修改了 EEPROM 设置](./raspberry-pi.mdx#更新-bootloader)，使 Raspberry Pi 5 设备在您按下电源按钮之前保持关机状态，否则情况即如此。

我们建议使用官方的树莓派电源。

不同的树莓派型号需要不同的功率才能正常运行。下表列出了各型号所需的 USB 电源传输模式。

:::

#### 连接电源
在为树莓派通电之前，您必须将操作系统 (OS) 镜像刷写（写入）到设备的存储介质中，否则树莓派将无法启动。更多信息，请参阅 [启动介质](#启动介质)。

配置好启动介质、将其插入树莓派并连接所有必要的外设后，将电源适配器插入树莓派上标有 **POWER IN**、**PWR IN** 或 **PWR** 的端口。某些树莓派型号（如 Zero 系列）的 USB 输出端口与电源端口具有相同的接口规格。请使用树莓派上的正确接口。随后，将电源线的另一端插入电源插座并接通电源。

此时树莓派将启动，电源或状态指示灯亮起即为启动标志。随后会显示带有树莓派徽标的信息屏幕。接下来，树莓派将从启动介质加载其操作系统。

## 网络连接 {#网络连接}
下表列出了树莓派设备的网络连接规格。

如果您的树莓派设备没有以太网端口，您仍可通过 USB 转以太网适配器连接有线网络。

同样地，如果您的树莓派设备没有内置 Wi-Fi，您仍可通过 USB 转 Wi-Fi 适配器连接无线网络。

*内置网络连接功能的树莓派型号*

| 型号名称 | 无线网络 | 以太网 |
| --- | --- | --- |
| Raspberry Pi  3B+ 及更新型号 | 是。双频 Wi-Fi（2.4/5 GHz） | 是。树莓派 3B+ 支持 300 Mb/s，树莓派 4B 及更新型号支持千兆以太网 |
| Raspberry Pi  3B | 是，仅支持 2.4 GHz | 是 || 树莓派 3A+ | 是。双频 Wi-Fi（2.4/5 GHz） | 不支持 |
| Raspberry Pi  1B、1B+ 和 2B | 不支持 | 支持 |

| Compute Module 5 (CM5) 和计算模块 4 (CM4) | 可选双频 Wi-Fi (2.4/5 GHz) | 是。千兆以太网。 |

## 设置您的树莓派 {#设置您的树莓派}
树莓派通常有两种使用方式：

* *作为台式电脑。* 这是一个带有桌面环境的交互式系统。此设置需要显示器、连接树莓派与显示器的线缆、键盘和鼠标。有关这些配件的信息，请参阅 [设备](#设备)。
* *作为无显示器计算机使用。* 这是一种无需传统外设（显示器、键盘和鼠标）即可运行的系统。您可以通过 Raspberry Pi Connect 访问，或通过 Wi-Fi 或以太网使用 SSH 从同一网络上的另一台计算机进行访问。在安装操作系统时，您可以预先配置主机名、用户账户、网络连接和 SSH 访问权限。


### 标准（桌面）设置 {#desktop-setup}

总体而言，树莓派的桌面设置包括启动介质、外设和电源，需按此顺序进行。网络连接为可选。

#### 步骤 1. 准备启动介质

使用 *Raspberry Pi Imager** 将操作系统安装到启动介质上。

这是使用树莓派的前提条件，也是您首次配置其大部分关键设置的环节，包括主机名、本地化设置、用户凭据、Wi-Fi 连接和远程访问。

其他设置可在后续进行配置。具体操作指南请参阅 [将操作系统安装到启动介质](#将操作系统安装到启动介质上)。

将操作系统安装到启动介质后，将其连接到树莓派。在较新型号中，这通常是将一张 microSD 卡插入设备底部的插槽中。如果您将操作系统安装在其他存储设备上，可以使用其他兼容的连接器将其连接到树莓派。

#### 步骤 2. 连接外设

若要将树莓派用作台式电脑，您可以连接以下外设：

* [输入外设](#输入外设)：
  * **键盘。** 对于使用旗舰版树莓派或树莓派 Zero 的桌面电脑而言是必需的；若您的树莓派是键盘电脑（树莓派 400、500 或 500+），则无需键盘。
  * **鼠标。** 对于桌面电脑而言是必需的。
* [输出外设](#输出外设)：
  * **显示器。** 桌面电脑必备。
  * **音频设备。** 可选。

<img src="/img/computers/getting-started/peripherals/cable-all.png" alt="将电源适配器连接至树莓派。" width="80%" />
*将外设连接至树莓派*

#### 步骤 3. 连接电源
连接好外设后，将电源适配器插入树莓派的 USB-C 电源接口（旧款机型为 micro USB）。

请确保电源适配器符合您所用树莓派型号的电压和电流要求。更多信息请参阅 [电源](#电源)。

当树莓派通电时，状态指示灯会亮起；如果树莓派已连接显示器，则会出现启动画面。此过程可能需要几分钟。

#### 步骤 4. 连接网络（可选）
在将操作系统安装到启动介质并插入树莓派后，您可以设置网络连接。这可以是有线（以太网）连接，也可以是无线（Wi-Fi）连接。

##### 连接无线网络
连接无线网络通常在启动介质配置过程中进行。有关配置启动介质的说明，请参阅 [将操作系统安装到启动介质](#将操作系统安装到启动介质上)。您也可以稍后通过桌面或命令行配置无线网络。具体操作请参阅 **配置** 页面中的 [连接到无线网络](./configuration.mdx#连接到无线网络)。

某些无线网卡和部分树莓派型号不支持 5 GHz 网络；请访问链接：[树莓派产品页面](https://www.raspberrypi.com/products/)，查看您的无线模块信息，以确保其与您首选的网络兼容。有关哪些树莓派型号支持无线网络的信息，请参阅 [网络连接](#网络连接)。

##### 连接有线网络
要将树莓派连接到有线网络，请将网线插入树莓派设备的以太网端口，直到听到 “咔嗒” 一声。如果您的树莓派型号未配备以太网端口，可以使用 USB 以太网适配器。

将网线的另一端连接到网络集线器、交换机或路由器上的端口。树莓派操作系统会自动连接到网络。

要拔出以太网线，请按住连接器底部的卡扣，然后将线缆轻轻从端口中滑出。

<img src="/img/computers/getting-started/peripherals/cable-net.png" alt="将以太网线插入树莓派。" width="80%" />
*将以太网线插入树莓派*

### 无显示器（远程）设置 {#无显示器设置}

无显示器的树莓派无需显示器、键盘或鼠标即可运行。

您可通过网络进行远程控制。为此，您必须具备：

* 网络连接（以太网或 Wi-Fi）。
* 设备首次启动时，使用 SSH 或 [Raspberry Pi Connect](../services/connect.mdx) 进行远程访问。后续启动时，您可以使用 [VNC](./remote-access.mdx#使用-vnc-共享屏幕)。请注意，VNC 不兼容 Raspberry Pi OS 的 Lite 版本。

我们建议您使用 **Raspberry Pi Imager** 来设置无显示器设备。这允许您在 **Raspberry Pi OS** 和 **Raspberry Pi OS Lite** 之间进行选择，并配置网络、远程访问以及用户凭据（用户名和密码）。

VNC 和 **Raspberry Pi Connect** 既可在桌面环境中启用，也可通过交互式 `raspi-config` 工具启用。具体操作指南请参阅 [远程访问](./configuration.mdx#远程访问)。

### 使用 SSH 设置无显示器的 Raspberry Pi

在开始步骤 1 之前，请注意以下事项：

* 部分无线网卡和树莓派型号不支持 5 GHz 网络，因此请查阅您的无线模块文档，以确保与您首选的网络兼容。更多信息请参阅 [网络](#网络连接)。

* 由于无显示器设置没有桌面环境，我们建议在使用 **Raspberry Pi Imager** 准备启动介质时，选择 **Raspberry Pi OS Lite** 作为操作系统。

#### 步骤 1. 准备启动介质

第一步是准备一张已安装树莓派操作系统的可启动磁盘。

1. 请按照 [使用 *Raspberry Pi Imager* 安装操作系统到启动介质](./getting-started.mdx#将操作系统安装到启动介质上) 中的说明进行操作。

2. 将操作系统安装到启动介质后，将其插入树莓派，或通过兼容的连接器将其连接到树莓派。

#### 步骤 2（可选）。连接有线网络
接下来，设置网络连接，以便无显示器设备能与您的计算机通信。

如果您已在步骤 1 中使用 Imager 设置了 SSID 和无线网络详细信息，则可以跳过此步骤，直接进入步骤 3。

如果您在步骤 1 中未设置无线网络连接，请现在将无显示器设备连接到有线网络：

1. 将以太网线插入树莓派设备的以太网端口，直到听到咔嗒声。如果您的树莓派型号未配备以太网端口，请使用 USB 以太网适配器。

2. 将网线的另一端连接到路由器的端口。树莓派操作系统会自动连接到网络。

<img src="/img/computers/getting-started/peripherals/cable-net.png" alt="将以太网线插入树莓派。" width="80%" />
*将以太网线插入树莓派*

#### 步骤 3. 连接电源

继续操作前，请确认您的电源适配器符合树莓派型号的电压和电流要求。更多信息请参阅 [电源](#电源)。

1. 将电源适配器插入树莓派的 USB-C 接口（旧款机型为 micro USB）电源端口。

当树莓派通电时，状态指示灯会亮起。此过程可能需要几分钟。

#### 步骤 4. 访问树莓派
首次启动时，唯一的远程访问选项是 SSH 或 Raspberry Pi Connect。

如果您在步骤 1 中未使用 Raspberry Pi Imager 自定义镜像，请手动设置远程访问。

在启动介质的 `bootfs` 分区上：

1. 创建一个名为 `ssh` 的空文件。
2. 创建一个名为 `userconf.txt` 的文件，内容仅包含一行文本：`<username>:<enrcypted-password>`。

请将 `<username>` 替换为您的用户名，并将 `<enrcypted-password>` 替换为密码的 *加密* 形式。

* **用户名**。您的用户名必须以字母开头，且仅可包含小写字母、数字、下划线和连字符。用户名最长为 31 个字符。
* **加密密码**。请在另一台计算机上使用 [OpenSSL](https://www.openssl.org) 生成加密密码。在命令行中运行 `openssl passwd -6`，随后在提示时输入您的密码并进行确认。该命令将输出所提供密码的加密版本。

有关详细信息，请参阅 [使用 SSH 访问远程终端](./remote-access.mdx#使用-ssh-访问远程终端)。


## 将操作系统安装到启动介质上 {#将操作系统安装到启动介质上}

要使用树莓派，您必须先将 **操作系统 (OS)** 安装（刷写）到存储设备上，使其成为启动设备。默认情况下，树莓派设备会检查插入 SD 卡槽中的任何 SD 卡是否含有操作系统。某些树莓派型号允许您从其他存储设备启动操作系统，例如 USB 驱动器、通过 HAT 连接的存储设备以及网络存储。以下说明假设您使用的是 microSD 卡。

我们建议您在另一台计算机上使用 **Raspberry Pi Imager** 将操作系统安装到存储设备上。有关说明，请参阅 [使用Imager安装](#使用-imager-安装)。如果您无法使用其他计算机将操作系统镜像写入存储设备，也可以尝试直接通过互联网在树莓派上安装操作系统。具体操作指南请参阅 [网络安装](#网络安装)。

### 例外情况
树莓派 500 以及部分树莓派桌面版和个人电脑套件自带一张预装了树莓派操作系统的 microSD 卡。树莓派 500+ 还包含一块预装了树莓派操作系统的 SSD。在这些情况下，您无需遵循本页的安装说明。不过，我们建议您将预装的操作系统更新至最新版本。具体操作指南请参阅 [更新当前树莓派软件](./os.mdx#update-software)。

若需重新配置树莓派，可通过连接至树莓派的显示屏顶部菜单进行操作。例如，您可以通过右上角的 Wi-Fi 图标设置或更改无线网络。具体操作指南请参阅 *配置* 页面中的 [连接到无线网络](./configuration.mdx#连接到无线网络)。

### 先决条件
要在存储设备上安装操作系统，您需要：

* 一个用作启动介质的空白存储设备，通常是 microSD 卡。有关更多信息，请参阅 [启动介质](#启动介质)。
* 一台可用于将操作系统镜像写入存储设备的计算机，以便树莓派能够从该设备启动。
* 将存储设备连接至该计算机的方法。

### 使用 Imager 安装 {#使用-imager-安装}

Raspberry Pi Imager 是一款可在 Mac、Windows 和 Linux 系统上用于下载并写入树莓派操作系统镜像的工具。

Imager 包含许多适用于树莓派设备的流行操作系统镜像，并支持加载从 [树莓派](https://www.raspberrypi.com/software/operating-systems/) 或 [Ubuntu](https://ubuntu.com/download/raspberry-pi) 等第三方供应商处手动下载的镜像。

在使用 Imager 进行配置时，您还可以预先配置树莓派设备的凭据和远程访问设置。

#### 步骤 1. 安装并启动 Imager

在您的计算机（Windows、Mac 或 Linux）上下载并安装 Raspberry Pi Imager。

1. 通过以下任一方式安装 Imager：
* 在 Windows 和 Mac 上，从 [软件下载](/downloads/) 下载最新版本并运行安装程序。
* 在 Raspberry Pi OS 上，运行 `sudo apt install rpi-imager`。
* 在其他 Linux 操作系统上，从 [软件下载](/downloads/) 下载最新版本的 AppImage，并将其设为可执行文件。

2. 通过以下任一方式启动 Raspberry Pi Imager 应用程序：
* 在 **应用程序** 菜单中找到 Imager。
* 点击计算机桌面上的 Raspberry Pi Imager 图标。
* 在命令行中以 root 身份运行 AppImage 文件。例如，使用以下命令：`sudo ./Raspberry_Pi_Imager-<version>-desktop-x86_64.AppImage`，其中 `<version>` 是一个数字，代表最新的 Imager 版本。

#### 步骤 2. 配置基础设置

请告知 Raspberry Pi Imager 您希望使用的硬件与操作系统组合：

1. 在 **设备** 选项卡中，从列表中选择您的 Raspberry Pi 型号，然后点击 **下一步**。
<img src="/img/computers/getting-started/imager/device-tab.png" alt="Raspberry Pi Imager 设备选项卡。" width="80%" />

2. 在 **操作系统** 选项卡中，选择一个操作系统，然后点击 **下一步**。

您可以选择安装现有的操作系统（列表顶部显示的是您所用树莓派型号的推荐操作系统），或者选择 *使用自定义*，从计算机上的镜像文件安装操作系统。
<img src="/img/computers/getting-started/imager/os-tab.png" alt="Imager 中的操作系统选择。" width="80%" />

3. 将存储设备连接到计算机。例如，使用外置或内置 SD 卡读卡器将 microSD 卡插入计算机。

4. 在 **存储** 选项卡中，选择您的存储设备，然后点击 **下一步**。

如果计算机上连接了多个存储设备，请务必选择正确的存储设备。通常可以通过容量来识别正确的存储设备。如果不确定，请断开其他设备的连接，直到确定要制作镜像的目标设备。

若您要写入的操作系统支持自定义，将打开 *自定义* 选项卡。

<img src="/img/computers/getting-started/imager/storage-tab.png" alt="Imager 中的存储选择选项。" width="80%" />

:::warning
我们强烈建议您保持 *排除系统驱动器* 选项勾选。这可防止您永久擦除数据并导致主机计算机无法运行。
:::

建议您现在按照 [自定义](#自定义操作系统) 中的说明进行操作。如果您不想现在进行，首次启动时系统会再次要求您提供相同的信息。对于 Raspberry Pi OS，此首次启动配置将在 [配置向导](#配置向导) 中进行。

#### 步骤 3. 自定义操作系统（推荐） {#自定义操作系统}

Raspberry Pi OS 及部分第三方操作系统支持自定义设置。

我们强烈建议您现在就设置您的 Raspberry Pi，但您也可以选择 **跳过自定义** 来跳过此步骤。

如果您现在不配置 Raspberry Pi，操作系统会在首次启动后再次要求您提供相同的信息。

如果您正在安装支持自定义的操作系统，可以预先配置：

1. 设备主机名
2. 时区
3. 用户名和密码
4. Wi-Fi 凭据
5. 远程连接
6. Raspberry Pi Connect

如果您已通过 Imager 完成了自定义步骤，其中某些设置的值可能已被填入。

要自定义操作系统，请打开 Imager 的 **自定义** 选项卡，然后按照以下各子选项卡的说明操作：

##### 主机名

在 **主机名** 子选项卡中，为您的树莓派输入一个仅包含字母、数字和连字符的名称，然后选择 **下一步**。

您的树莓派将通过 [mDNS](https://en.wikipedia.org/wiki/Multicast_DNS) 向网络广播此主机名。当您将树莓派连接到网络时，该网络上的其他设备可以使用`<hostname>.local`或`<hostname>.lan`与您的树莓派进行通信。

<img src="/img/computers/getting-started/imager/hostname-subtab.png" alt="Imager 操作系统自定义 - 主机名选项卡。" width="80%" />

##### 本地化

在 **本地化** 子选项卡中，选择您的首都城市。Imager 会自动补全该城市的时区和键盘布局。您也可以单独更改这些设置。

您的选择还会设置 Wi-Fi 监管域；监管域始终与所选首都相对应。

完成后，请选择 **下一步**。

<img src="/img/computers/getting-started/imager/localisation-subtab.png" alt="Imager 操作系统自定义 - 本地化选项卡。" width="80%" />

##### 用户
在 **用户** 子选项卡中，输入树莓派管理员用户的用户名和密码，然后选择 **下一步**。

<img src="/img/computers/getting-started/imager/user-subtab.png" alt="Imager OS 自定义 - 用户选项卡。" width="80%" />

用户名必须为小写，且仅包含字母、数字、下划线和连字符。

* 对于无显示器设置，树莓派会在首次启动时使用这些凭据连接到网络。

在此阶段创建用户名和密码并不意味着每次启动树莓派时都需要输入这些凭据。若需在启动时要求输入密码，请关闭 **自动登录**。具体操作请参阅 [启动时要求输入密码](#自动登录)。

##### 自动登录 {#自动登录}

##### Wi-Fi
在 **Wi-Fi** 子选项卡中，选择连接 **加密网络** 还是 **开放网络**。在此之前，如果您使用的是 Mac 电脑，系统可能会询问您是否希望从系统钥匙串中预填 Wi-Fi 密码。请确认是否执行此操作，并使用您的 Mac 电脑凭据登录。

首次使用时，Imager 通常会预填您当前连接的 Wi-Fi 网络的 SSID（名称）和密码。如果没有预填信息，或者您想更改它，请输入相应无线网络的 SSID。对于安全网络，请输入并确认该网络的密码。如果您的网络未公开广播 SSID，请启用 **隐藏 SSID** 设置，以便操作系统能够找到该网络。完成后选择 **下一步**。

如果您正在配置无显示器的树莓派，请务必在此时设置 Wi-Fi 连接，因为树莓派需要在启动的瞬间就知道如何连接网络。

<img src="/img/computers/getting-started/imager/wifi-subtab.png" alt="Imager OS 自定义 Wi-Fi 选项卡。" width="80%" />

##### 远程访问
在 **远程访问** 子选项卡中，配置 SSH 设置，然后选择 **下一步**。

<img src="/img/computers/getting-started/imager/ssh-subtab.png" alt="Imager OS 自定义 远程访问 选项卡。" width="80%" />

若要在网络上远程使用树莓派，请通过相应的开关启用 **启用 SSH**。

如果您计划运行无显示器的树莓派（而非将其作为台式电脑使用），您可以选择开启 SSH，或为设备配置 **Raspberry Pi Connect**。

如果启用 SSH，请选择以下其中一种身份验证方法：

* **使用密码身份验证**：通过网络使用您在 **自定义 > 用户** 选项卡中提供的用户名和密码登录树莓派。
* **使用公钥认证**：将树莓派配置为使用基于 SSH 密钥的认证。Imager 会显示一个字段供您输入公钥。如果您在 SSH 配置中已有 RSA 公钥，Imager 将使用该公钥。若没有，您可以将公钥复制到 Imager 中，或点击 **浏览** 在计算机上选择公钥文件。

##### Raspberry Pi Connect
某些操作系统还会显示 **Raspberry Pi Connect** 子选项卡。要将设备与您的 Raspberry Pi Connect 账户关联，请完成以下步骤：

1. 使用切换按钮开启 **启用 Raspberry Pi Connect**。
<img src="/img/computers/getting-started/imager/connect-subtab.png" alt="Imager 操作系统自定义 Raspberry Pi Connect 选项卡。" width="60%" />

2. 选择 **打开 Raspberry Pi Connect**。这将在您的默认网页浏览器中打开 Raspberry Pi Connect。
<img src="/img/computers/getting-started/imager/os-customisation-site.png" alt="Raspberry Pi Connect 页面。" width="60%" />

3. 登录您的 Raspberry Pi ID 账户；如果您还没有账户，请注册。
4. 在 **新建授权密钥** 页面中，创建您的授权密钥。这是一个仅限一次性使用的临时令牌。Raspberry Pi Connect 网站会显示该令牌自创建起过期的时间。要使用该令牌，请在过期时间之前启动您的 Raspberry Pi 并将其连接到互联网。
<img src="/img/computers/getting-started/imager/connect-auth-key.png" alt="Raspberry Pi Connect 新认证密钥页面。" width="60%" />

    * 如果您属于一个或多个组织，请选择要关联该密钥的组织或账户。然后选择 **创建认证密钥并启动 Raspberry Pi Imager**。 

5. 您的浏览器可能会询问您是否允许该网站打开 Raspberry Pi Imager。请确认您要打开 Raspberry Pi Imager。
    * Imager 将打开至 **Raspberry Pi Connect** 选项卡，该选项卡会显示一条消息，确认 Imager 已从浏览器接收到了认证令牌，并显示一个包含该令牌的字段。
    * 如果令牌未被自动复制，您可以手动将其复制到 Imager 中。打开 Raspberry Pi Connect 网页上的 **遇到问题？** 部分，复制令牌，然后将其粘贴到 Imager 中的令牌字段中。
6. 选择 **下一步** 继续。

#### 第 4 步：设置 Imager 选项
您可以配置 Imager 在写入过程中的行为。在 Imager 界面的左下角选择 **应用选项**，然后在“应用选项”窗口中使用下列各选项旁边的开关来启用或禁用它们：

* **写入完成后播放声音**：配置 Imager 在写入完成后播放提示音。
* **写入完成后弹出存储介质**：配置 Imager 在写入完成后自动卸载存储设备。
* **启用匿名统计数据遥测**：配置 Imager 在写入完成后发送有关您的设备和操作系统的匿名使用统计数据。有关更多信息，请参阅 [匿名指标（遥测）](https://github.com/raspberrypi/rpi-imager?tab=readme-ov-file#anonymous-metrics-telemetry)。
* **禁用警告**：配置 Imager 在写入过程中抑制警告消息。

高级用户还可以设置自定义 **内容存储库**，从中获取操作系统和设备列表。有关更多信息，请参阅 [Imager 存储库 URL](./raspberry-pi.mdx#IMAGER_REPO_URL)。

完成设置后，请在 **应用选项** 窗口的右下角选择 **保存**。

<img src="/img/computers/getting-started/imager/app-options.png" alt="应用选项对话框。" width="60%" />

#### 步骤 5. 将镜像写入存储设备

浏览完 Imager 中的各个选项卡后，您可以在 *写入* 选项卡中查看所选设置的 *摘要*。

<img src="/img/computers/getting-started/imager/write.png" alt="在写入前于 Imager 中查看选项。" width="80%" />

如果您对选项满意并希望继续：

1. 在 Imager 右下角选择 **写入**。
2. Imager 会显示提示，警告您即将覆盖存储介质上的所有数据。若要继续，请选择 **我已了解，擦除并写入**。若出现管理员提示，要求授予读写存储介质的权限，请授予 Imager 相应权限以继续操作。
<img src="/img/computers/getting-started/imager/are-you-sure.png" alt="在 Imager 中确认对存储设备进行重映像操作。" width="80%" />

3. Imager 开始将镜像写入存储设备。此过程可能需要几分钟。
4. 写入镜像后，Imager 会验证镜像是否已正确写入。此过程同样可能需要几分钟。我们建议您完成验证步骤。但如果您想取消验证，请选择 **跳过验证**。
5. 镜像写入并验证完成后，Imager 将显示 **完成** 选项卡。选择 **完成** 以退出 Imager。

现在，您可以按照 [标准（桌面）设置](#desktop-setup) 或 [无显示器（远程）设置](#无显示器设置) 中的说明，根据您的需求开始使用树莓派。


### 网络安装
您可以使用通过网络下载的 Raspberry Pi Imager 版本，将操作系统安装到存储设备上。使用 **网络安装** 功能时，除树莓派本身外，无需额外的 SD 卡读卡器或计算机。您可以在任何兼容的存储设备（包括 SD 卡和 USB 存储设备）上运行网络安装。

如果您的树莓派运行的是旧版引导加载程序，您可能需要更新引导加载程序才能使用网络安装。具体操作指南请参阅 **配置** 页面中的 [更新引导加载程序](./configuration.mdx#update-bootloader-version)。

#### 先决条件
网络安装需要以下条件：

* 运行支持 **网络安装** 固件的兼容树莓派型号。网络安装仅在树莓派 4B 及后续旗舰型号以及树莓派键盘电脑（400、500 和 500+）上运行。
* 物理连接到树莓派设备的显示器和键盘。
* 有线互联网连接。
* 连接至树莓派设备的空白兼容存储设备，例如 SD 卡或 USB 闪存盘。这是网络安装将操作系统镜像写入的目标位置。如果存储设备上已安装操作系统（或任何可引导系统），且您未按住 **Shift** 键，树莓派将正常启动，而非进入网络安装模式。

#### 启动网络安装

1. 将显示器和键盘连接到树莓派设备，然后执行以下操作之一：
    * 将一个空的（不可引导的）存储设备（如 SD 卡或 USB 闪存盘）连接到树莓派设备，

或者

    * 在为树莓派通电时，按住连接到树莓派的键盘上的 **Shift** 键。
    <img src="/img/computers/getting-started/network-install-1.png" alt="网络安装界面。" width="80%" />

2. 若尚未连接，请使用网线将树莓派连接至互联网。连接成功后，树莓派将自动下载安装程序。若下载失败，请重复上一步骤尝试。
<img src="/img/computers/getting-started/network-install-2.png" alt="开始网络安装。" width="80%" />

3. 请等待安装程序下载完成。
<img src="/img/computers/getting-started/network-install-3.png" alt="通过网络安装下载 Imager。" width="80%" />

4. 当 Raspberry Pi 安装程序下载完成后，您的 Raspberry Pi 会自动加载 Raspberry Pi Imager。请按照 [使用Imager安装](#使用-imager-安装) 中的说明操作。
<img src="/img/computers/getting-started/network-install-4.png" alt="选择存储设备。" width="80%" />

有关网络安装配置的更多信息，请参阅 [HTTP 启动](./raspberry-pi.mdx#http启动)。


## 配置

Raspberry Pi 最基本的配置通常在 Imager 中的操作系统安装过程中完成。

### 首次启动时的配置 {#配置向导}

本节介绍安装后的配置，无论您：

* 跳过了 Imager 中的自定义步骤。
* 未使用 Imager 安装操作系统。
* 正在使用预装了操作系统的 Raspberry Pi，或者
* 希望在安装后更改设置，以根据您的需求定制树莓派。

如果您按照 [将操作系统安装到启动介质](#将操作系统安装到启动介质上) 中的 **步骤3** 所述，在 Imager 的操作系统自定义过程中完成了树莓派的预配置，那么除非您想更改任何设置，否则无需再次配置树莓派。

如果您采用的是 **桌面设置** 且在 Imager 中跳过了操作系统自定义步骤，或者您使用的是预装了操作系统的树莓派，则首次启动时会显示配置向导。

**无显示器设置** 会跳过配置向导；由于您已在 Imager 中预先配置了设置，您的无显示器树莓派将直接启动并连接到网络。如果您选择了 SSH 进行远程访问，可以在计算机的命令行中输入 `ssh username@hostname.local` 来远程连接到无显示器的树莓派。有关更多信息，请参阅 [无显示器（远程）设置](#无显示器设置)。

## 关机选项
树莓派操作系统提供了通过桌面和命令行安全关机、重启、锁定以及注销的选项。

### 桌面关机选项
桌面上的 **关机选项** 窗口提供以下选项：

* **关机**。关闭所有正在运行的程序并关闭树莓派电源。在断开树莓派电源前请使用此选项。
* **重启**。重新启动系统。此选项在应用更新、更改配置以及连接新硬件后非常有用。
* **锁定屏幕**。锁定当前会话以防止未经授权的访问，同时保持已打开的程序继续运行。输入密码即可再次解锁屏幕。
* **注销**。退出当前用户账户并关闭所有正在运行的程序，但不会关闭系统。

### 从桌面打开关机选项
要在桌面打开 **关机选项** 窗口，请选择系统托盘左上角的树莓派图标，然后选择 **关机**。

以下设备也提供关机快捷方式：

* Raspberry Pi 5：双击电源按钮以关闭设备（**关机选项**窗口可能会短暂显示）。
* Raspberry Pi 5、500 或 500+：单击 **电源按钮** 以打开 **关机选项** 窗口。
* Raspberry Pi 400：按下 **Fn** + **F10** 打开 **关机选项** 窗口。

<img src="/img/computers/getting-started/shutdown.png" alt="从桌面菜单关机。" width="40%" />
*桌面关机选项*

<img src="/img/computers/getting-started/shutdown-menu.png" alt="关机菜单。" width="40%" />
*桌面关机菜单*

您可以使用鼠标，或键盘方向键和 **Enter** 键，从以下四个选项中进行选择：关机、重启、锁屏和注销，具体说明请参见 [桌面关机选项](#桌面关机选项)。

若不选择任何选项直接退出菜单，请点击右上角的叉号（**×**）图标或按下 **Esc** 键。

### 命令行关机选项
您可通过命令行执行大部分关机操作。

打开终端，然后使用以下任一命令：

| 操作 | 命令 |
| --- | --- |
| 关闭树莓派 | `sudo poweroff` |
| 重启树莓派 | `sudo reboot` |
| 在控制台模式下，不关闭系统直接注销 | `logout` |

## 故障排除

如果树莓派在 5 分钟内无法启动，请检查状态指示灯。如果状态指示灯在闪烁，请查阅 [LED 警告闪烁代码](./configuration.mdx#led警告闪烁代码) 来诊断问题。

如果树莓派仍然无法启动，且 LED 信息无法帮助您解决问题，请尝试以下方法：

* 如果您使用了 SD 卡以外的启动设备，请尝试从 SD 卡启动。
* 按照 [使用 Imager 安装](#使用-imager-安装) 中的说明重新写入 SD 卡，并完整完成 **验证** 步骤。
* 在主机计算机上使用 Imager，对存储设备执行 [重新刷写引导加载程序](./raspberry-pi.mdx#更新-bootloader)，然后执行 [重新写入 SD 卡映像](#使用-imager-安装)。

安装并配置好树莓派后，您可以下载软件，开始使用您的全新树莓派电脑，并加入更广泛的社区以获取灵感和支持。

### 推荐应用程序
树莓派操作系统（Raspberry Pi OS）预装了多款应用程序，让您能够尽快开始使用树莓派。

您还可以免费下载其他推荐应用程序：点击屏幕左上角的树莓派图标，然后选择 **首选项 > 推荐软件**。在此处，您可以安装各种推荐应用程序。

例如，如果您计划将树莓派用作家用电脑，LibreOffice 对于撰写和编辑文档及电子表格会非常有用。您还可以通过 **通用访问** 下的 Orca 屏幕阅读器，让树莓派更易于使用。
