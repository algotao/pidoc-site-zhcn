## 安装

下载体积较大，可将源[设置为国内镜像](../../pidoc/edit_apt_source.mdx)，可大大提升下载速度。

如果对编译安装不熟练，尽量使用系统的 apt 命令安装模式，以避免编译时的各种异常。

<Tabs>
<TabItem value="cpp" label="C++安装">

```sh
sudo apt install libopencv-dev cmake
```

</TabItem>

<TabItem value="python" label="Python安装">

```sh
sudo apt install libopencv-dev python3-opencv
```

</TabItem>
</Tabs>
