### GPIO Zero
GPIO Zero 库是树莓派官方推荐的用于操作树莓派 GPIO 的 Python 库，这是当前在 树莓派5 及 早期型号 经过测试的 GPIO 库。它对常用外设的操作进行了封装，使用起来非常方便。2.0版本其底层GPIO库使用了lgpio，使得能最大可能兼容后续的树莓派的GPIO变化。
