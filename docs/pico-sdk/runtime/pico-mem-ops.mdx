提供编译器内置 memcpy、memset 及相关函数的优化替换实现。

### 详细描述

包含以下函数：

* memset, memcpy

* `__aeabi_memset, __aeabi_memset4, `__aeabi_memset8, __aeabi_memcpy`, __aeabi_memcpy4, __aeabi_memcpy8`

此库不提供任何额外函数。
