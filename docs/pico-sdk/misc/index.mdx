# Miscellaneous

本章包含以下 API 分组：

- `boot_picobin_headers`(boot-picobin-headers.mdx): Constants for PICOBIN format.
- `boot_picoboot_headers`(boot-picoboot-headers.mdx): Header file for the PICOBOOT USB interface exposed by an RP2xxx chip in BOOTSEL mode.
- `boot_uf2_headers`(boot-uf2-headers.mdx): Header file for the UF2 format supported by a RP2xxx chip in BOOTSEL mode.
- `pico_usb_reset_interface_headers`(pico-usb-reset-interface-headers.mdx): Definition for the reset interface that may be exposed by the pico_stdio_usb library.
