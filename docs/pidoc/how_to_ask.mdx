# 如何向网友有效提问

## 提问前先阅读

QQ/微信等交流工具提供了快速便捷的交流方式，但如果遇到问题，建议遵循以下步骤。

最终能否得到有用的回答，往往**取决于你提问和追问的方式**。

**问题解决步骤**：
1. **尝试自身解决**：运用你掌握的知识先尝试自己解决问题。不要被英文输出信息吓倒，尽量读懂它。绝大多数信息输出都有指向性，你只需要找到它，然后自己尝试解决。

2. **在网上搜索**：在网上搜索一下，看看有没有人遇到过类似的问题。如果有，请先参考一下别人的解决方案，如果没有，再在群里提问。

3. **在群里提问**：在群里提问，请先描述清楚问题，然后附上截图，方便大家快速定位问题。你准备的信息越丰富，别人解决问题的效率就越高。

### 存在的问题
- **随手拍屏**。能截图就不要拍屏，请学会使用各种工具附带的截图功能。如因为场景特殊无法截屏（如在boot期间），此种情况下拍屏才是可接受的。
- **信息不完整**。编程细分领域庞杂。仅编程语言就有几十种。将您的问题限定在特定场景中，给出具体条件，有助于聚焦问题。

## 有效提问

请花一点时间整理问题，描述清楚以下信息：

1. **什么硬件设备**。例如：树莓派3B+XX摄像头
2. **什么操作系统**。例如：树莓派OS Desktop Bookworm，32位
3. **什么编程语言**。例如：Python 3.9.6
4. **什么问题现象**。例如：调用xxxx时，报错xxxxx。无法获取摄像头输出内容。附完整信息截图
5. **附上实现代码**。
6. **如何复现**：
   1. 插上电源
   2. 开机
   3. 运行XXX
   4. 报错
