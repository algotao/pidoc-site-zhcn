# 使用国内源加速您的 APT

::::warning
本文档适用于 Raspberry Pi OS 的 Trixie(13)/Bookworm(12) 版本。

**Trixie(13) 与 Bookworm(12) 的 APT 源文件格式不同：**
- **Trixie(13)**：采用新的 DEB822 格式，文件名为 `debian.sources` 和 `raspi.sources`，存储在 `/etc/apt/sources.list.d/` 目录下。
- **Bookworm(12)**：沿用传统的 one-line 格式，文件名为 `sources.list` 和 `raspi.list`。
::::

## 快速实施

使用本站作为镜像源，一键替换。

<Tabs groupId="osversion">
<TabItem value="trixie" label="Trixie (13)">

```sh
# Trixie 使用 DEB822 .sources 格式，替换 URI 即可
sudo sed 's|URIs: http://deb.debian.org|URIs: http://mirrors.pidoc.cn|g' /etc/apt/sources.list.d/debian.sources -i
sudo sed 's|URIs: http://archive.raspberrypi.com/debian/|URIs: http://mirrors.pidoc.cn/raspberrypi/|g' /etc/apt/sources.list.d/raspi.sources -i
sudo apt update
```

</TabItem>
<TabItem value="bookworm" label="Bookworm (12)">

```sh
sudo sed 's/deb.debian.org/mirrors.pidoc.cn/g' /etc/apt/sources.list -i
sudo sed 's/archive.raspberrypi.com\/debian\//mirrors.pidoc.cn\/raspberrypi\//g' /etc/apt/sources.list.d/raspi.list -i
sudo apt update
```

</TabItem>
</Tabs>

## 手动修改

如果您想手动修改，请按照以下步骤操作。

<Tabs groupId="osversion">
<TabItem value="trixie" label="Trixie (13)">

:::note
Trixie 使用新的 **DEB822 格式** 的 `.sources` 文件，内容为多行键值对结构。
:::

#### 1. 备份 `debian.sources` 文件
```sh
sudo cp /etc/apt/sources.list.d/debian.sources /etc/apt/sources.list.d/debian.sources.bak
```

#### 2. 编辑 `debian.sources`
```sh
sudo nano /etc/apt/sources.list.d/debian.sources
```

#### 3. 将原始 Debian 源地址替换为本站源地址

原始内容：
```
Types: deb
URIs: http://deb.debian.org/debian
Suites: trixie trixie-updates
Components: main contrib non-free non-free-firmware
Signed-By: /usr/share/keyrings/debian-archive-keyring.gpg

Types: deb
URIs: http://deb.debian.org/debian-security/
Suites: trixie-security
Components: main contrib non-free non-free-firmware
Signed-By: /usr/share/keyrings/debian-archive-keyring.gpg
```

替换后（只需修改每段中的 `URIs` 字段）：
```
Types: deb
URIs: http://mirrors.pidoc.cn/debian
Suites: trixie trixie-updates
Components: main contrib non-free non-free-firmware
Signed-By: /usr/share/keyrings/debian-archive-keyring.gpg

Types: deb
URIs: http://mirrors.pidoc.cn/debian-security/
Suites: trixie-security
Components: main contrib non-free non-free-firmware
Signed-By: /usr/share/keyrings/debian-archive-keyring.gpg
```

> 保存并关闭文件。在 nano 编辑器中，按下 `Ctrl + X`，然后按 `Y` 确认保存，最后按 `Enter` 退出。

#### 4. 备份并编辑 `raspi.sources`
```sh
sudo cp /etc/apt/sources.list.d/raspi.sources /etc/apt/sources.list.d/raspi.sources.bak
sudo nano /etc/apt/sources.list.d/raspi.sources
```

原始内容：
```
Types: deb
URIs: http://archive.raspberrypi.com/debian/
Suites: trixie
Components: main
Signed-By: /usr/share/keyrings/raspberrypi-archive-keyring.gpg
```

替换后：
```
Types: deb
URIs: http://mirrors.pidoc.cn/raspberrypi/
Suites: trixie
Components: main
Signed-By: /usr/share/keyrings/raspberrypi-archive-keyring.gpg
```

</TabItem>
<TabItem value="bookworm" label="Bookworm (12)">

:::note
Bookworm 沿用传统的 **one-line 格式** 的 `.list` 文件，每行一条源配置。
:::

#### 1. 备份 `sources.list` 文件
```sh
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
```

#### 2. 编辑 `sources.list`
```sh
sudo nano /etc/apt/sources.list
```

#### 3. 将原始 Debian 源地址替换为本站源地址

原始内容：
```
deb http://deb.debian.org/debian bookworm main contrib non-free non-free-firmware
deb http://deb.debian.org/debian-security/ bookworm-security main contrib non-free non-free-firmware
deb http://deb.debian.org/debian bookworm-updates main contrib non-free non-free-firmware
```

替换后：
```
deb http://mirrors.pidoc.cn/debian bookworm main contrib non-free non-free-firmware
deb http://mirrors.pidoc.cn/debian-security/ bookworm-security main contrib non-free non-free-firmware
deb http://mirrors.pidoc.cn/debian bookworm-updates main contrib non-free non-free-firmware
```

> 保存并关闭文件。在 nano 编辑器中，按下 `Ctrl + X`，然后按 `Y` 确认保存，最后按 `Enter` 退出。

#### 4. 备份并编辑 `raspi.list`
```sh
sudo cp /etc/apt/sources.list.d/raspi.list /etc/apt/sources.list.d/raspi.list.bak
sudo nano /etc/apt/sources.list.d/raspi.list
```

原始内容：
```
deb http://archive.raspberrypi.com/debian/ bookworm main
```

替换后：
```
deb http://mirrors.pidoc.cn/raspberrypi/ bookworm main
```

</TabItem>
</Tabs>

#### 5. 更新 APT 源列表
```sh
sudo apt update
```
