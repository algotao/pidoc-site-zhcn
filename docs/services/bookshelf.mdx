# 书架（Bookshelf）

## 简介

树莓派操作系统中的书架应用程序提供树莓派出版社出版的书籍和杂志的访问权限。自树莓派操作系统Trixie版本起，该应用还包含仅限贡献者访问的出版物。

* 免费书籍以云朵图标标识，任何用户均可下载。
* 贡献者专属书籍以挂锁图标标识，除非您使用关联贡献记录或纸质杂志订阅的 [树莓派ID](id.mdx) 登录。部分贡献者专属书籍后续将向全体用户开放。

<img src="/img/services/bookshelf/bookshelf-with-padlock.png" alt="书架" width="80%" />

Bookshelf 软件已包含在 Raspberry Pi OS Desktop 及 Raspberry Pi OS Full 镜像中。

## 成为捐赠者（Raspberry Pi OS Trixie 及更高版本）

仅限贡献者查看的标题对通过 [树莓派ID](id.mdx) 登录 Bookshelf 的用户开放，该 ID 需关联以下任一事项：

* 持有 [订阅我们的印刷版杂志](https://raspberrypipress.imbmsubscriptions.com/)邮箱地址的用户
* [月度捐赠](https://magazine.raspberrypi.com/bookshelf/link)

设置月度捐赠步骤：

1. 在 Bookshelf 左下角选择**贡献...**，跳转至树莓派官网贡献者页面。
1. 若尚未拥有树莓派ID，可立即创建。已有账户请使用用户名和密码登录。.
1. 注册月度捐赠计划。.
1. 点击**登录Bookshelf**以开启该应用，并存储您的捐赠者身份标识ID。

此时**捐赠...**按钮将消失，所有刊物均可下载。

您可使用树莓派ID登录，在任意数量的设备上访问可用书籍。若更换新树莓派设备或刷写不同版本的树莓派操作系统，只需重新登录即可再次下载心仪书籍。

## 下载书籍

双击任意可用书籍即可下载PDF文件，并通过默认PDF阅读器打开。

若您停止月度捐赠，已下载的书籍仍可保留。这些PDF文件存储于您主目录下的**Bookshelf**文件夹中，并可通过Bookshelf持续访问。
